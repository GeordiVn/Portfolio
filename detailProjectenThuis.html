Android Applicaties

Gerechten app

Ik heb een android applicatie gemaakt voor de gerechten die ik samen met mijn vriendin maak in op te slagen. Je kan hier nieuwe gerechten in toevoegen, recepten linken aan het gerecht en nog een aantal opties. In deze app kunnen we ook een weekmenu samenstellen waar een boodschappenmenu aanhangt. Alles wordt opgeslagen op mijn server thuis en we kunnen alles opvragen waar we ook zijn.

Kalender app

Deze android applicatie heb ik ook zelf gemaakt. Dit is een agenda waar je al je taken/afspraken kunt inplannen. Je hebt dan ook een tab planning waar je kan zien wat je dagplanning is. Deze app heb ik gemaakt voor het vak Project management. De applicatie houd alles bij op mijn server thuis zodat alles ook gesynchroniseerd kan worden met mijn andere toestellen.

File manager app

Hier heb ik een file manager applicatie gemaakt waar ik mijn foto's en bestanden in kan opslagen. Deze bestanden en foto's worden opgeslagen op mijn server thuis. Zodus ik kan op elk apparaat met deze applicatie aan mijn bestanden aan. Heel handig want je kan ook rechtstreeks een foto maken die snel wordt opgeslagen in de huidige locatie waar je staat.

Experimenten en uitvindingen

Prototype inductie verwarming

Ik heb meestal het probleem als ik aan de computer zit met een koffie of thee dat ik deze even vergeet en dat het koud wordt. Nu met dit apparaatje is dit probleem opgelost. Door een roestvrij stalen plaatje op de boden van mijn tas blijft mijn inhoud warm. Dit komt doordat ik met een inductie verwarming werk. Hier plaats je je tas op de orangje spoel dat een snel wisselend elektromagnetisch veld uitstraald. Het inox plaatje wordt warm omdat het in dat veld terecht komt en door eddy stromen die kortsluiten in het plaatje wordt het warm.
