Java programma

Buddy client programma

Voor het vak Object georienteerd programmeren moeten we als project een java swing applicatie maken. Ik heb gekozen voor een chat programma te maken dat ik BuddyChat noem. Het programma heeft de mogelijkheid om met meerdere gebruikers te praten (al gesimuleerd met 1000 clients). Dit gebreurd met sockets over tcp zodat je over heel de wereld kan chatten met elkaar.

Buddy server programma

Het server programma dient voor de aanvragen te verwerken van de clients die geconnecteerd zijn. Via tcp sockets komen aanvragen en berichten binnen. De server verwerkt de aanvraagen en berichten en zet het in een csv log bestand. Ook scant de server continu of dat de client nog geconnecteerd is met de server zodat andere gebruikers kunnen zien wie online is.
